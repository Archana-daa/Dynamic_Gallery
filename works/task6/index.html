GitHub Setup and Task 5 Submission Guide

1. Create and Setup your GitHub account

If you don't have a GitHub account, go to https://github.com/ and sign up.

2. Create a new repository for the class assignments

Create a new repository on GitHub for your class assignments. You can either:

  • Create a public repository, or
  • Collaborate with https://github.com/aum-web-app

3. Pull the repository to your local machine

Clone the repository to your local machine using the git clone command. Open your terminal or command prompt and navigate to the directory where you want to store your project. Then, use the following command:

git clone <repository_URL>

Replace <repository_URL> with the URL of your repository.

4. Create a branch for Task 5: Calculator assignment

Create a new branch for your Task 5 assignment. Use the following command:

git branch Task5_Calculator

5. Checkout to the newly created branch

Switch to the newly created branch using the git checkout command:

git checkout Task5_Calculator

6. Copy Task 5 (Calculator) files to your local repository

Copy your Task 5 (Calculator) files into your local repository within the Task5_Calculator branch.

7. Stage all changes

Stage all the changes you've made using the git add command:

git add .

8. Push changes to remote with appropriate commit message

Commit your changes with a descriptive message and push them to the remote repository.

git commit -m "Added Task 5 Calculator files" git push origin Task5_Calculator

9. Make pull request and accept merge request

Create a pull request on GitHub to merge your Task5_Calculator branch into the main (or master) branch. Once the pull request is reviewed and approved, merge it.

10. Submit your repository URL and commit ID in Canvas

Submit the following information in Canvas:

  • Repository URL: The URL of your repository (e.g., https://github.com/Archana-daa/Task5_calculator_DA.git)
  • Commit ID: The commit ID of the merge commit (the commit that merged your Task 5 branch into main or master).

11. Checkout to the main branch

Switch back to the main branch (main or master) using the git checkout command:

git checkout main or git checkout master
